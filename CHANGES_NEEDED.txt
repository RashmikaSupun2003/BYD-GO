================================================================================
                    CHANGES NEEDED BEFORE RUNNING THE APP
================================================================================

1. SUPABASE CONFIGURATION
   File: config/supabase.ts
   
   Replace these values with your Supabase project credentials:
   - supabaseUrl: "YOUR_SUPABASE_URL"
   - supabaseAnonKey: "YOUR_SUPABASE_ANON_KEY"
   
   How to get: Supabase Dashboard → Project Settings → API

2. GOOGLE MAPS API KEY (iOS)
   File: app.json
   Line: ~59
   
   Replace: "YOUR_IOS_GOOGLE_MAPS_API_KEY"
   With: Your actual iOS Google Maps API key

3. GOOGLE MAPS API KEY (Android)
   File: app.json
   Line: ~78
   
   Replace: "YOUR_ANDROID_GOOGLE_MAPS_API_KEY"
   With: Your actual Android Google Maps API key

4. SUPABASE TABLES SETUP
   - Create 'charging_stations' table in Supabase
   - Create 'favorites' table in Supabase
   - Set up Row Level Security (RLS) policies
   - See Supabase dashboard for table schemas

================================================================================
                            SETUP STEPS
================================================================================

1. Run: npm install

2. Set up Supabase:
   - Create project at https://supabase.com/
   - Create 'charging_stations' table
   - Create 'favorites' table
   - Copy config to config/supabase.ts

3. Set up Google Cloud:
   - Create project at https://console.cloud.google.com/
   - Enable: Maps SDK for Android, Maps SDK for iOS, Places API, Geocoding API
   - Create API keys
   - Update app.json and component files

4. Run: npm start
   Then press 'a' for Android or 'i' for iOS
